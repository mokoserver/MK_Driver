# GIPROSVJAZ_Driver

Этот каталог содержит драйверы для проекта GIPROSVJAZ, разработанные с использованием National Instruments LabVIEW.

## Содержимое

- **DriverCompiled_DLL**: Скомпилированные библиотеки (DLL) драйвера.
- **Drivers**: Исходный код драйверов.
- **PythonDocs**: Документация для использования драйверов с Python.
- **Файлы проекта LabVIEW** (`.lvproj`, `.lvlps`, `.aliases`): Файлы, необходимые для работы с проектом в среде LabVIEW.
- **info.txt**: Текстовый файл, возможно, содержащий дополнительную информацию.

Этот драйвер обеспечивает взаимодействие программного обеспечения с аппаратной частью проекта GIPROSVJAZ.
